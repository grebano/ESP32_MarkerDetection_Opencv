# ESP32_MarkerDetection_Opencv
The goal is to detect square markers in a photo taken by the esp32 that also runs opencv, creating a standalone device.

## Hardware
- ESP32-CAM (AI-Thinker)
- 5V to 3.3V step down converter
- 5V power supply
- 3D printed case
- 9V flash light
- 9V power supply

## Software
- Espressif IDF
- Opencv C++
